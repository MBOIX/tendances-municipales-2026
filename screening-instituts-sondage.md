## 4. Méthodologie d'agrégation des sondages

### 4.1 Méthode FiveThirtyEight (référence mondiale)

#### Pondération par taille d'échantillon

```
poids_échantillon = sqrt(n_sondage) / sqrt(n_médiane)
```

Rendements décroissants (racine carrée). Normalisé par rapport à la taille médiane.

#### Pondération par récence (EWMA)

```
poids_récence = exp(-decay × sqrt(jours_depuis_sondage))
```

Moyenne mobile pondérée exponentiellement. Le paramètre `decay` détermine la vitesse d'obsolescence.

#### Pondération par note du sondeur (POLLSCORE)

```
POLLSCORE = f(précision_historique, méthodologie, transparence)
```

Combine erreur historique et biais dans un score agrégé.

#### Déduplication temporelle

Si un sondeur publie **N sondages en 14 jours**, chaque sondage reçoit un poids de `1/N`.

---

### 4.2 Formules recommandées pour ce dashboard

#### Formule de base : moyenne pondérée

```
Score_agrégé(candidat) = Σ(score_i × w_i) / Σ(w_i)
```

Où le poids `w_i` de chaque sondage `i` combine :

```
w_i = w_récence × w_échantillon × w_fiabilité × w_dédup
```

#### Composantes :

**Récence** (demi-vie ~7 jours pour les municipales) :
```
w_récence = exp(-0.1 × jours)
```

**Taille d'échantillon** (normalisé à 1 pour n=1000) :
```
w_échantillon = sqrt(n) / sqrt(1000)
```

**Fiabilité de l'institut** :
```
w_fiabilité = score_institut / 10
```

**Déduplication** :
```
w_dédup = 1 / nombre_sondages_même_institut_14_jours
```

#### Intervalles de confiance

```
marge_erreur (95%) = 1,96 × sqrt(p×(1-p)/n)
marge_agrégée ≈ marge_individuelle / sqrt(k)
```

Où `k` = nombre effectif de sondages indépendants après déduplication.

## Annexes

### A. Biais connus dans les sondages français

1. **Surestimation du RN en ligne** : le biais d'autocensure a disparu et s'est inversé
2. **Sous-estimation de LFI** : électorats jeunes et populaires difficiles à capter
3. **Vote utile de dernière minute** : non captable (cas Mélenchon 2022)
4. **Biais de panel en ligne** : ~40% de la population exclue des panels Internet
5. **Recettes secrètes** : méthodes de redressement non publiées en détail

### B. Cadre réglementaire

- **Loi du 19 juillet 1977** : cadre légal des sondages d'opinion
- **Commission des sondages** : autorité indépendante de contrôle
- **Période de réserve** : interdiction de publier des sondages pendant les 48h précédant chaque tour
- **Sanction** : amende de 75 000 € pour violation
